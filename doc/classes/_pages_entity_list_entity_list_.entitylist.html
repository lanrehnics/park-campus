createEntity

  • createEntity(): void
    • Defined in pages/entity-list/entity-list.ts:61

    Creates a modal, passing in the entity, which allows the user to create an entity of that type, such as a lecture or a module. Upon dismissal of the modal page, the entity is created using the relevant provider. Chained promises are used to create a lecture due to the asynchronous nature of the process involved.

    Returns void
